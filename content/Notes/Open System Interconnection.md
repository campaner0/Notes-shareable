The OSI was the first standard model for computer networks, developed in the 80s. It replaced the monopoly IBM had on computer networking.

7 layer networking 
1. Physical
2. Data Link
3. Network
4. Transport
5. Session
6. Presentation
7. Application

[[TCP]]/[[IP]] function on the transport/network level
